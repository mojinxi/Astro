# 安装青龙



首先，你得有台小鸡，至于哪里搞，随你，你问我哪里买的？我的是racknerd家的，3台机子一年9刀，现在是传家宝，绝版了😁



想装青龙就得装docker，docker的安装方法我之前文章有<a href='https://mojinxi.cn/3.html'>记一次用Docker部署Aria2-pro</a>，不想看的可以直接看<a href='https://docs.docker.com/engine/install/debian'>官方文档</a>，翻到官方文档最后，有一键脚本。



docker装好后就简单了，直接把下面代码扔进去，等待安装完成。在浏览器输入你的ip地址加上5700（下面代码里自己可以改，建议更改）就行了，例如192.168.1.1:5700

```

docker run -dit \

  -v $PWD/ql/config:/ql/config \

  -v $PWD/ql/log:/ql/log \

  -v $PWD/ql/db:/ql/db \

  -v $PWD/ql/repo:/ql/repo \

  -v $PWD/ql/raw:/ql/raw \

  -v $PWD/ql/scripts:/ql/scripts \

  -p 5700:5700 \

  --name qinglong \

  --hostname qinglong \

  --restart unless-stopped \

  whyour/qinglong:latest

```

这个时候青龙就装好了，然后你就可以看我这篇文章<a href='https://mojinxi.cn/6.html'>《京东签到（一）》</a>，只看拉库那里就行了，这个时候只要放入你的cookie，就可以进行 ~~ 薅羊毛 ~~ 签到了，当然，一般懒人是不会手抓cookie的，那么就进行下一步。



# 安装nvjdc



其实这个更简单，说简单是因为有一键安装脚本，而非一键脚本的教程我懒得写🙄一键脚本来自<a href='https://github.com/buqian123/Tasks'>翔翎大佬</a>（大佬删库了）

```

bash <(curl -sL https://raw.githubusercontents.com/buqian123/Tasks/main/onekey-install-nvjdc.sh)

```

上面的只更新到1.4，现在用下面的吧

```

bash <(curl -sL https://raw.githubusercontent.com/07031218/normal-shell/main/onekey-install-nvjdc.sh )

```

安装就很简单了

- 请输入青龙服务器在web页面中显示的名称：你的青龙服务器要显示的名称（随便填）

- 请输入nvjdc面板标题：网站标题（随便填）

- 请输入nvjdc面板希望使用的端口号：5600（这个和青龙的5700差不多，看喜好）

- XDD面板和toyken不启用，我们这个教程只是青龙和nvjdc

- 然后对接青龙，这里说下OpenApiClientID和OpenApiClientSecret，如下图所示，在青龙里添加应用即可

![](https://dl2.img.timecdn.cn/2021/11/27/0DE4E59F-81CF-45D6-82F7-A829C922CCF2.jpg)

- 然后青龙url就是你的服务器ip和5700，例如http://192.168.1.1:5700

- 安装完成后会有访问地址，进入后手机号获取验证码就好
